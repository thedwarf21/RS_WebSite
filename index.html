Le jeu Asteroids Harvest

Asteroids Harvest est un jeu HTML5 open source développé from scratch par nos soins à titre de démonstration.

Il s'agit d'une revisite du jeu Asteroids édité par Atari en 1979 et maintes fois cloné depuis lors. Asteroids Harvest propose également :

  • La possibilité d'améliorer le vaisseau, via un magasin s'ouvrant avant chaque niveau.
  • Lorsqu'un astéroïde est touché par un tir, sa trajectoire est modifiée selon l'angle et la puissance du tir.
  • Un système de sauvegarde par cookie.
  • Possibilité d'afficher les hitbox.
  • Sensibilité radiale paramétrable.

Le code source est accessible publiquement sur la plateforme GitHub.
